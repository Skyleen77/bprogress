---
title: set()
description: BProgress.set() method
---

The `set()` method is used to set the progress bar to a specific percentage.

## Usage

```js
BProgress.set(0.5);
```

## Typing

```ts
BProgress.set(n: number): BProgress;
```

## Parameters

| Name | Type   | Description          |
|------|--------|----------------------|
| n    | number | The percentage value between 0 and 1 |
