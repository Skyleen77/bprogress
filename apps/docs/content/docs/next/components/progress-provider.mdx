Import the correct `ProgressProvider` according to your Next.js configuration.

## Import

```tsx
// App Directory
import { AppProgressProvider as ProgressProvider } from '@bprogress/next';
// Pages Directory
import { PagesProgressProvider as ProgressProvider } from '@bprogress/next';
```

## Usage

```tsx
<ProgressProvider>
  ...
</ProgressProvider>
```
