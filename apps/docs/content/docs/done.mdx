---
title: done()
description: BProgress.done() method
---

The `done()` method is used to stop the progress bar.

## Usage

```js
BProgress.done();
```

## Typing

```ts
BProgress.done(force?: boolean): BProgress;
```

## Parameters

| Name  | Type    | Description                                                        |
|-------|---------|--------------------------------------------------------------------|
| force | boolean | Force the progress bar to finish immediately, even if not started. |
