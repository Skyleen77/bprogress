Next NProgress Bar is a simple progress bar for Next.js applications. It is lightweight and easy to use. It is inspired by the NProgress library.

## Installation

To install Next NProgress Bar, run the following command:

<Tabs groupId="packageManager" items={['npm', 'pnpm', "yarn", "bun"]} persist>
  <Tab value="npm">
    ```bash
    npm install next-nprogress-bar
    ```
  </Tab>
  <Tab value="pnpm">
    ```bash
    pnpm add next-nprogress-bar
    ```
  </Tab>
  <Tab value="yarn">
    ```bash
    yarn add next-nprogress-bar
    ```
  </Tab>
  <Tab value="bun">
    ```bash
    bun add next-nprogress-bar
    ```
  </Tab>
</Tabs>

Now you are ready to use Next NProgress Bar in your Next.js application!

<Callout title="Note">Next NProgress Bar is based on **nprogress-v2** which is a more modern and functional version of **nprogress**.</Callout>
